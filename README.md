# Turtlesim Catch Them All

A foundational autonomous navigation project built with **ROS 2 Jazzy** and the turtlesim simulation package. The custom `turtle_controller` node uses a proportional (P) control loop to calculate distance and angular errors, driving the main turtle to autonomously "catch" the closest active turtle. New targets are continuously generated by a custom `turtle_spawner` node. The entire system, including custom interfaces and node communication, is orchestrated using a single launch file.


### Pre-requisite

> 1. Ubuntu 24.04 LTS (Recommended for Jazzy)
> 2. ROS 2 Jazzy installed

### Follow the steps

1. Navigate to your ROS 2 workspace source folder.

```bash
cd ~/ros2_ws/src 
```

2. Clone the repository.

```bash
git clone "https://github.com/Nilupul-Seram/turtlesim_catch_them_all_project.git" 
```

3. Navigate back to the root of your workspace and build the project.

```bash
cd ~/ros2_ws
colcon build --packages-select my_robot_interfaces turtlesim_catch_them_all my_robot_bringup
```
4. Source the setup file.

```bash
source install/setup.bash
```
5. Launch the simulation.

```bash
ros2 launch catch_them_all_bringup catch_them_all.launch.xml
```

🛠️ VS Code IntelliSense Configuration (C++)

If you are using Visual Studio Code, IntelliSense might show errors when trying to include your custom interfaces. To fix this, you need to point VS Code to your workspace's install directory.

Open `.vscode/c_cpp_properties.json` in your workspace and add the path `"~/ros2_ws/install/catch_them_all_project_interfaces/include/**"` to the generated headers inside the "includePath" array:

```JSON
{
    "configurations": [
        {
            "browse": {
            "databaseFilename": "${workspaceFolder}/.vscode/browse.vc.db",
            "limitSymbolsToIncludedHeaders": false
        },
            "includePath": [
                "${workspaceFolder}/**",
                "/opt/ros/jazzy/include/**",
                "~/ros2_ws/install/catch_them_all_project_interfaces/include/**"
            ],
            "name": "ros2",
            "compilerPath": "/usr/bin/gcc",
            "cStandard": "c17",
            "cppStandard": "gnu++17",
            "intelliSenseMode": "linux-gcc-x64"
        }
    ],
    "version": 4
}
```
